# Active Directory Automation Scripts

This folder contains PowerShell scripts designed to automate common Active Directory administrative tasks in enterprise environments.

## 🔧 Overview

The scripts in this directory focus on improving efficiency, consistency, and accuracy when managing users, computers, and group memberships at scale.

## 🚀 Key Capabilities

- Bulk computer group assignment for application and access provisioning  
- Bulk user attribute lookup and reporting  
- Active Directory computer OU (Organizational Unit) discovery and validation  
- Endpoint placement auditing and directory hygiene checks  

## 🧠 Use Cases

- Standardizing workstation provisioning across environments  
- Validating user and device information in Active Directory  
- Auditing endpoint placement and organizational structure  
- Reducing manual administrative effort through automation  

## ⚙️ Technologies Used

- PowerShell  
- Active Directory Module (`ActiveDirectory`)  
- LDAP queries and AD object manipulation  

## 🔒 Disclaimer

All scripts have been sanitized for public sharing.  
No sensitive, proprietary, or environment-specific information is included.
