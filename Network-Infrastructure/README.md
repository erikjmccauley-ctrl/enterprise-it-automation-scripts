# Network Infrastructure Scripts

This folder contains PowerShell scripts used for network-level visibility, troubleshooting, and infrastructure management.

## 🚀 Key Capabilities

- DHCP scope identification  
- IP address lease and reservation validation  
- Network troubleshooting and diagnostics  

## 🧠 Use Cases

- Investigating IP conflicts  
- Verifying DHCP assignments  
- Identifying reserved vs active IP usage  
- Supporting network and endpoint troubleshooting  

## ⚙️ Technologies Used

- PowerShell  
- DHCP Server module  
- IP parsing and subnet evaluation  

## 🔒 Disclaimer

All scripts have been sanitized for public sharing.  
No sensitive, proprietary, or environment-specific information is included.
