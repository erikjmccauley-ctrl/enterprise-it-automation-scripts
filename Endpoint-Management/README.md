# Endpoint Management & Remediation Scripts

This folder contains PowerShell scripts used for remote endpoint maintenance, troubleshooting, and system remediation in enterprise environments.

## 🚀 Key Capabilities

- Remote cache and temporary file cleanup  
- Endpoint troubleshooting and remediation  
- System restart and maintenance automation  
- SCCM cache and Group Policy reset workflows  

## 🧠 Use Cases

- Resolving software deployment issues  
- Clearing corrupted cache or temporary files  
- Forcing policy refresh scenarios  
- Improving endpoint performance and stability  

## ⚙️ Technologies Used

- PowerShell  
- Remote administrative shares  
- Windows system directories  
- Endpoint management tooling (SCCM/Intune environments)  

## 🔒 Disclaimer

All scripts have been sanitized for public sharing.  
No sensitive, proprietary, or environment-specific information is included.
