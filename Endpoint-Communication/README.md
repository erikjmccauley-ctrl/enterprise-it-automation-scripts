# Endpoint Communication Scripts

This folder contains scripts used to send messages and notifications to users on remote endpoints in enterprise environments.

## 🚀 Key Capabilities

- Bulk user notification across multiple endpoints  
- Remote message delivery using built-in Windows tools  
- Rapid communication for IT support and operational updates  

## 🧠 Use Cases

- Notifying users of required IT actions  
- Communicating service desk instructions  
- Alerting users during troubleshooting or maintenance  

## ⚙️ Technologies Used

- Windows Batch scripting  
- `msg` command for remote communication  

## 🔒 Disclaimer

All scripts have been sanitized for public sharing.  
No sensitive, proprietary, or environment-specific information is included.
